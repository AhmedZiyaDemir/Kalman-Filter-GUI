# Kalman Filter Signal Analyzer

## Overview
This project is a Python-based Graphical User Interface (GUI) application designed to analyze and filter dynamic signals using the Kalman Filter algorithm. It provides an interactive environment to visualize noisy data and demonstrate the smoothing and prediction capabilities of the filter.

## Features
* **Interactive GUI:** Developed using PyQt5 for a seamless and user-friendly experience.
* **Mathematical Filtering:** Implements the Kalman Filter to process and smooth noisy data sequences.
* **Dynamic Visualization:** Utilizes Matplotlib to render high-quality plots, comparing raw signals with the filtered output.

## Technologies & Libraries
* Python
* PyQt5
* Matplotlib
* NumPy
